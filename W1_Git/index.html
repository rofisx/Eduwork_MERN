Summary Git Learning

  1. Git Command
    • git init : Untuk menginisialisasi git di repo
    • git add . : Untuk staging file sebelum commit
    • git commit -m"" : Untuk commit hasil perubahan file
    • git reset --hard : Untuk mengembalikain perubahan ke commit terkahir
  2. Git Branch
    • git branch : Untuk melihat semua branch
    • git branch (nama branch) : untuk membuat branch baru
    • git checkout (nama branch) : untuk pindah ke branch yang dituju
    • git branch -d (nama branch) : untuk menghapus branch
  3. Git GUI
    • Clone reporsiroty git dari gitlab.com
    • Solving conflict dengan source tree
    • Merge file dengan GUI sorce tree
